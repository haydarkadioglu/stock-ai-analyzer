How to Get Gemini API Key

Step-by-step guide to get your Gemini API key from Google AI Studio

1

Visit Google AI Studio

Go to Google AI Studio and sign in with your Google account.

Open Google AI Studio
2

Create API Key

Click on "Create API Key" button and select "Create API key in new project" or choose an existing project.

3

Copy Your API Key

Once created, your API key will be displayed. Copy it immediately as it won't be shown again.

4

Paste and Save

Go to Settings page, paste your API key in the input field and click "Save Settings". You can test it before saving using the "Test" button.

Go to Settings

Important Note

Security: Keep your API key secure and never share it publicly. You can revoke and regenerate API keys from Google AI Studio if needed.
